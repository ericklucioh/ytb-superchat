Como funciona na prática

O processo foi pensado para ser simples: você abre os chats, o painel reúne as mensagens e você decide o que quer destacar durante a live.

1

Abra os chats popout

Deixe as páginas de chat abertas para que as mensagens possam ser acompanhadas normalmente.

2

A extensão observa o chat

Ela identifica quando chegam mensagens novas e envia essas informações para o painel.

3

As informações ficam com você

O conteúdo é mantido no navegador para que a experiência continue leve e direta.

4

O painel unifica tudo

A interface mostra os chats em uma única visualização, mais fácil de acompanhar durante a transmissão.

5

Gere overlays

Selecione uma mensagem e leve esse destaque para o overlay da live ou para o OBS.

Fluxo direto, sem complicação desnecessária.

A proposta é deixar o caminho entre o chat e o painel o mais curto possível, para que você veja e use as mensagens sem depender de etapas extras.

Com isso, a transmissão fica mais previsível, mais leve e mais fácil de controlar no dia a dia.

O que cada parte faz

  • Captura → acompanha as mensagens nas plataformas suportadas
  • Controle → organiza o funcionamento da extensão em segundo plano
  • Sincronização → mantém a extensão e o painel conversando entre si
  • Painel → exibe, filtra e separa as mensagens que importam

Privacidade e processamento local

O projeto foi pensado para dar mais controle a você sobre o que entra no painel e como isso aparece na live.

Nada é enviado para fora

O uso cotidiano não exige mandar os chats para serviços externos só para o painel funcionar.

Somente mensagens públicas

A leitura acontece nas páginas públicas de chat que você já usa na transmissão.

Sem coleta pessoal

O foco é mostrar mensagens ao vivo, sem transformar o projeto em uma ferramenta de coleta de dados.

Pontos importantes antes de usar

Alguns detalhes ajudam a entender melhor o que esperar do projeto no dia a dia.

Quais plataformas são suportadas?

No uso principal, o projeto foi preparado para YouTube, Twitch e Kick.

Pode quebrar com mudanças de layout?

Sim. Se a plataforma mudar muito a forma como mostra o chat, pode ser preciso ajustar a captura.

Usa API oficial?

Não. A ideia é ler o chat de forma local e prática, sem depender de integrações oficiais para funcionar.

Pronto para centralizar sua live?
