# Projeto: Plataforma de Centralização de Chats para Streamers Profissionais

## Objetivo
A extensão Chrome extrai mensagens dos popups de chat (YouTube, Twitch, Kick) e salva no storage local. O portal lê essas mensagens do storage. Ao clicar em uma mensagem, conecta na API, que sincroniza com o overlay (OBS). A API serve apenas para atualizar o overlay, não para centralizar dados do chat.

## Público-alvo
Streamers profissionais.

## Tecnologias
- Frontend: Estático com JavaScript Vanilla
- Extensão Chrome
- Backend: API simples em Go

## Escopo Inicial
Foco em melhorias e integrações na plataforma já existente, priorizando facilidade de integração com OBS.

## Observações
- Não considerar novas funcionalidades neste momento, apenas aprimoramentos e integrações.
