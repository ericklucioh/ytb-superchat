# 01-03-PLAN: Event API & Integration

## Objetivo
Implementar API para ingestão de eventos e integração com chats das plataformas.

## Tarefas
- Criar rota /api/event para receber eventos do site
- Encaminhar eventos para o hub WebSocket (overlay)
- Testar ingestão e distribuição de eventos do site para o overlay

## Critérios de Aceite
- API recebe eventos das plataformas
- Eventos são distribuídos em tempo real
- Testes cobrem ingestão e distribuição
