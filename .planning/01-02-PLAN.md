# 01-02-PLAN: WebSocket Hub & Overlay Support

## Objetivo
Adicionar suporte a WebSocket para distribuição de eventos e servir arquivos estáticos para overlay.

## Tarefas
- Implementar hub WebSocket no backend Go
- Criar rota /ws para conexão
- Servir arquivos estáticos para overlay
- Testar comunicação em tempo real

## Critérios de Aceite
- Conexões WebSocket funcionam e distribuem eventos
- Overlay pode ser carregado via servidor
- Testes cobrem cenários de conexão e evento
