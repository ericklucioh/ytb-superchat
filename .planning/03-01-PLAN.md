# 03-01-PLAN: Dashboard & Extension Sync

## Objetivo
Garantir que o portal leia corretamente as mensagens do storage local preenchido pela extensão Chrome, e que ao clicar em uma mensagem, a API seja chamada para atualizar o overlay (OBS).

## Tarefas
- Garantir que a extensão Chrome extraia e salve corretamente as mensagens dos popups de chat no storage local
- Garantir que o portal leia e exiba corretamente as mensagens do storage local
- Implementar chamada à API ao clicar em uma mensagem para atualizar o overlay
- Testar integração ponta a ponta: extração → leitura → clique → overlay

## Critérios de Aceite
- Mensagens extraídas pela extensão aparecem no portal
- Clique em mensagem aciona atualização do overlay via API
- Testes cobrem o fluxo completo real
