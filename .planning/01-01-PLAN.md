# 01-01-PLAN: Foundation & Session Management

## Objetivo
Implementar o servidor HTTP ytb-go com rotas básicas e gerenciamento de sessão em memória.

## Tarefas
- Criar servidor HTTP em Go
- Implementar rotas /health e /api/session
- Gerenciar sessões em memória
- Escrever testes básicos para as rotas

## Critérios de Aceite
- Servidor responde corretamente nas rotas básicas
- Sessões são criadas, listadas e removidas em memória
- Testes cobrem os fluxos principais
