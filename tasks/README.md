2. `02-backend-go-overlay.md`

## Como Usar
- Cada arquivo descreve uma entrega independente.
- A ordem sugerida respeita dependências técnicas.
- As tarefas devem ser lidas como backlog executável, não como brainstorming.

## Critério De Aceite Do Backlog
- Toda tarefa tem objetivo, dependência, entrega e critério de pronto.
- O backlog cobre a aplicação até o estado pronto.
- Não sobra ambiguidade relevante para quem for executar.
